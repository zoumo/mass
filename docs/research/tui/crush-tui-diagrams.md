## 2. Dialog 栈系统

### 渲染层次 (从底到顶)

```
  ┌─ Layer 4 ─────────────────────────────────────┐
  │                                               │
  │   ╔═══════════════════════════════╗            │  ← Dialog Overlay (最顶层)
  │   ║  Commands              [×]   ║            │     / 命令面板
  │   ║  ┌─────────────────────────┐ ║            │     模型选择
  │   ║  │ 🔍 Type to filter...   │ ║            │     Session 切换
  │   ║  └─────────────────────────┘ ║            │
  │   ║  ▸ New Session           n   ║            │     只有这层接收键盘事件
  │   ║    Switch Session        s   ║            │
  │   ║    Switch Model          m   ║            │
  │   ╚═══════════════════════════════╝            │
  │                                               │
  ├─ Layer 3 ─────────────────────────────────────┤
  │                                               │
  │                    ┌─────────────────────┐    │  ← 补全弹窗
  │                    │ src/main.go         │    │     @ 触发文件/资源补全
  │                    │ src/handler.go      │    │     向上弹出
  │                    │ pkg/client/api.go   │    │
  │                    └─────────────────────┘    │
  │  > 请查看 @src/|                              │
  │                                               │
  ├─ Layer 2 ─────────────────────────────────────┤
  │  ctrl+c quit · tab focus · ctrl+p commands    │  ← Status Bar + 临时消息
  │                                               │
  ├─ Layer 1 ─────────────────────────────────────┤
  │                                               │
  │  ┃ Assistant 消息...                          │  ← 主界面
  │  ┃                                            │     Header + Chat + Editor
  │  > 输入区域_                                  │
  │                                               │
  └───────────────────────────────────────────────┘
```

### Dialog 栈操作

```
  初始状态               按 /                    再按 ctrl+l

  ┌───────────┐      ┌───────────┐           ┌───────────┐
  │           │      │           │           │ Model     │ ← 栈顶 (接收事件)
  │  主界面   │      │ Commands  │ ← 栈顶   ├───────────┤
  │           │      ├───────────┤           │ Commands  │ ← 挂起
  └───────────┘      │  主界面   │           ├───────────┤
                     └───────────┘           │  主界面   │
                                             └───────────┘
      空栈               1 个 Dialog              2 个 Dialog
                     键盘 → Commands           键盘 → Model Picker

  选择模型后 (Esc)       再 Esc

  ┌───────────┐      ┌───────────┐
  │ Commands  │ ← 恢复栈顶     │           │
  ├───────────┤      │  主界面   │
  │  主界面   │      │           │
  └───────────┘      └───────────┘
    1 个 Dialog           回到主界面
  键盘 → Commands
```

### 与 MASS 现状对比

```
  Crush (4 层交互)                    MASS 现状 (1 层)

  ╔═══════════════╗                   ┌───────────────┐
  ║  Dialog       ║ ← 命令发现       │               │
  ╠═══════════════╣                   │               │
  ║  补全弹窗     ║ ← @ 文件引用     │  Chat + Editor │ ← 只有这一层
  ╠═══════════════╣                   │  (无命令面板)  │
  ║  Status Bar   ║ ← 状态反馈       │               │
  ╠═══════════════╣                   │               │
  ║  主界面       ║ ← 聊天对话       ├───────────────┤
  ╚═══════════════╝                   │  Status Bar   │
                                      └───────────────┘
  功能可被发现                         需要记住快捷键
  临时消息有反馈                       无临时消息
  文件可 @ 引用                        无补全
```
